Graph Creatin via QGraphicsView


You can also create or change your graph using QGraphicsView field, which is on the left side of the interface

  • Double LMB click: Create the Node
  • Single LMB click: Select Node (will become blue)
  • Delete button: Delete all selected Nodes
  • Single RMB click on the Node: Open context menu

Options of context menu:

  • Set this as Source: sets this Node as Source (for Ford-Falkerson) or Start (for Deicstra). Node will become red
  • Set this as Sink: sets this Node as Sink (for Ford-Falkerson)
  • Change color
  • Change name
  • Link Selected to this: Connects selected Node with the one on wich action menu was called
  • Change weight: Changes weight of all Edges leading from selected Nodes to the one wich the context menu was called
  • Remove all connections: Removes all Edges leading from and to this Node
